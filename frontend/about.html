About Optivest

Modern Portfolio Theory Optimiser

What is Optivest?

Optivest is a tool designed to help investors build optimised portfolios using Markowitz's Modern Portfolio Theory (MPT). You can select assets, define a time period, and calculate expected returns, volatility, and the Sharpe ratio.

Key Features

  • Quickly add or remove asset tickers
  • Select historical periods for analysis
  • Optimise portfolios based on expected return and risk
  • Visualise results in a clean, card-style layout
  • Export or copy results for further analysis

Purpose

This tool is ideal for investors, students, and anyone interested in data-driven portfolio optimisation. It simplifies complex calculations and helps make informed investment decisions.

Mathematics Behind Optivest

  • Portfolio Expected Return:
    Weighted average of individual asset returns:
    E(Rp) = Σ wi * E(Ri)
  • Portfolio Volatility (Risk):
    Accounts for correlations between assets:
    σp = √ΣΣ wi * wj * σi * σj * ρij
  • Sharpe Ratio (Risk-Adjusted Return):
    Return per unit of risk:
    Sharpe Ratio = (E(Rp) - Rf) / σp
  • Optimisation (Modern Portfolio Theory):
    Maximise return for a given risk or minimise risk for a given return using quadratic programming.
  • Diversification:
    Combining low-correlation assets reduces overall portfolio risk and creates an efficient portfolio on the “efficient frontier.”
